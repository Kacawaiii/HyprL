# Beta Onboarding Checklist

## Goal

Onboard a new user in under 15 minutes with no automation.

## Steps

1) Confirm payment received.
2) Provision access:
   - Add Discord role or invite to private channel.
   - Issue API key if applicable.
3) Send onboarding packet:
   - `docs/PRODUCT_V0.md`
   - `docs/legal/DISCLAIMER.md`
   - Track record link.
4) Verify user understands:
   - Signals are generic and informational.
   - No execution on user accounts.
   - Risk remains with user.
5) Support contact:
   - Response target: within 24 hours.

## Recommended user guidance

- Start in paper mode.
- Use fixed risk per trade and hard limits.
- Do not overtrade in the first two weeks.
